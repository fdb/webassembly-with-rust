# WebAssembly with Rust

Explorations in WebAssembly with Rust, based on the book [Programming WebAssembly with Rust](Programming WebAssembly with Rust) by Kevin Hoffman.

## Installation (macOS)

```
brew install wabt
curl https://sh.rustup.rs -sSf | sh
```

## Testing
```
cd checkers
npm install
npm test
```
